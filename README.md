# Automated Payment Data Preprocessor

A Python-based automation tool built to streamline financial data preprocessing by automating manual filtering, sorting, and cleaning workflows.

## Features
- Automatically parses and sorts chronological data.
- Cleans records based on status, client exclusions, and action history.
- Outputs a minimized, task-ready Excel sheet.
- Built-in error handling for locked files, missing files, and schema mismatches.

## Technologies Used
- Python 3
- Pandas
- Openpyxl (Excel processing engine)
